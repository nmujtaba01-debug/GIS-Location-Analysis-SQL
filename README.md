# GIS Location Analysis with SQL 🗺️

Real SQL analysis of customer location data
using W3Schools Northwind database.

## Business Questions Answered

1. Total number of customers in database
2. Customers from cities starting with B
3. Cities with more than 1 customer
4. Customers with complete location data
5. Top 3 countries by customer count
6. Customers from countries starting with G or F
7. Countries with more than 3 customers
8. All customers sorted by Country then City

## SQL Skills Demonstrated

- SELECT with specific columns
- WHERE with LIKE pattern matching
- IS NOT NULL for data quality check
- GROUP BY for location grouping
- HAVING for filtered aggregation
- ORDER BY with multiple columns
- TOP for limiting results
- COUNT for customer counting

## Why This Project Matters

Location data analysis is a core skill in
GIS development. This project demonstrates
ability to extract spatial insights from
customer databases — a real job requirement
in GIS and SAP HANA roles.
